Описание

Данный репозиторий содержит полный код сайтя "Shady Rhymes" созданного для конвертации валют с помощью Django.

Установка

1-создание virtual environment: python -m venv venv

2-активация venv: venv\Scripts\activate

3-установка фреймворков: py -m pip install -r req.txt

Запуск

1-открытие папки: cd Shady_Rhymes

2-запуск проекта: py manage.py runserver

Регистрация superuser, для создания товаров

py manage.py createsuperuser
